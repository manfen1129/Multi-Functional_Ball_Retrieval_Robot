# 机器人控制系统

本项目拟使用Jetson Nano作为中控系统，搭配使用Arduino UNO（可选），对电机进行驱动。其中可以直接使用键盘来控制机器人，也可以使用PS2等遥控器进行控制。此外，为充分使用Jetson Nano优秀的性能，项目还研究了基于大模型的具身智能机器人的实现。
